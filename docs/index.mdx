# Memorose turns raw agent traffic into memory that compounds.

Memorose ingests events, consolidates facts and procedures, previews semantic updates or forgetting before execution, projects shared organizational knowledge, tracks goals and tasks, and exposes the whole runtime through a Rust server, a dashboard, and Raft-backed deployment model.

## Runtime map

| Layer | What it holds |
| --- | --- |
| `L0` | Raw events: text, image, audio, video, and JSON |
| `L1` | Stable memory: facts, preferences, procedural traces, and summaries |
| `L2` | Insights: themes, graph-linked clusters, and organization knowledge |
| `L3` | Goals and tasks: milestones, dependencies, progress, and outcomes |

## What ships now

- Stream-first HTTP API for ingest and retrieval
- Hybrid recall across vector, text, graph, time, and organization knowledge
- Semantic preview and execute flows for corrections and forgetting
- Goal and task runtime with trees, readiness, and progress tracking
- Dashboard APIs for memory inspection, graph views, and cluster operations
- Single-node and Raft-backed deployment modes

## Key routes

- `POST /v1/users/:user_id/streams/:stream_id/events`
- `POST /v1/users/:user_id/streams/:stream_id/retrieve`
- `GET /v1/users/:user_id/tasks/tree`
- `GET /v1/organizations/:org_id/knowledge`
- `POST /v1/users/:user_id/memories/semantic/preview`
- `POST /v1/users/:user_id/memories/semantic/execute`

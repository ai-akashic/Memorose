# Python SDK

The Memorose Python SDK provides a simple interface for interacting with your Memorose server.

## Installation

```bash
pip install memorose
```

## Quick Start

```python
from memorose import Memorose

client = Memorose(
    url="http://localhost:3000",
    api_key="your-api-key"
)
```

## Storing Memories

```python
# Simple
client.add(
    content="User's favorite color is blue",
    user_id="alice",
    app_id="my-app"
)

# With metadata and agent scoping
client.add(
    content="User completed onboarding flow",
    user_id="alice",
    app_id="my-app",
    agent_id="onboarding-bot",
    metadata={
        "source": "onboarding",
        "step": "complete",
        "importance": 0.9
    }
)
```

## Searching Memories

```python
results = client.search(
    query="What are the user's preferences?",
    user_id="alice",
    app_id="my-app",
    limit=10,
    min_score=0.5
)

for memory in results:
    print(f"[{memory.score:.2f}] {memory.content}")
    print(f"  Metadata: {memory.metadata}")
```

## Deleting Memories

```python
client.delete(memory_id="mem_abc123")
```

## Listing Memories

```python
memories = client.list(
    user_id="alice",
    app_id="my-app",
    limit=50
)
```

## LangChain Integration

```python
from memorose.integrations import MemoroseLangChainMemory

memory = MemoroseLangChainMemory(
    client=client,
    user_id="alice",
    app_id="my-app"
)

# Use with any LangChain agent
agent = create_agent(llm=llm, memory=memory)
```

## Error Handling

```python
from memorose.exceptions import MemoroseError, NotFoundError

try:
    client.delete("nonexistent_id")
except NotFoundError:
    print("Memory not found")
except MemoroseError as e:
    print(f"API error: {e.code} - {e.message}")
```

# Getting Started

Welcome to Memorose — the open-source, self-hosted long-term memory layer for AI agents.

## Quick Install

### Docker (Recommended)

```bash
docker compose up -d
```

This starts Memorose on port 3000 with the dashboard at port 3000/dashboard.

### Binary

```bash
curl -fsSL https://memorose.dev/install.sh | sh
memorose --config config.toml
```

### From Source

```bash
git clone https://github.com/memorose/memorose.git
cd memorose
cargo build --release
./target/release/memorose-server
```

## Python SDK

Install the Python SDK:

```bash
pip install memorose
```

## Your First Memory

```python
from memorose import Memorose

client = Memorose("http://localhost:3000")

# Store a memory
client.add(
    content="User prefers dark mode and Python",
    user_id="alice",
    app_id="my-app",
    metadata={"source": "preferences"}
)

# Search memories
results = client.search(
    query="What does the user prefer?",
    user_id="alice",
    limit=5
)

for memory in results:
    print(f"{memory.content} (score: {memory.score:.2f})")
```

## Configuration

Create a `config.toml`:

```toml
[server]
host = "0.0.0.0"
port = 3000

[storage]
data_dir = "./data"

[embedding]
model = "all-MiniLM-L6-v2"
dimensions = 384
```
